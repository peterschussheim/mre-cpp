## Getting Started

### Requirements

- C++11 compatible compiler: g++ (version >= 4.2), or clang++ (version >= 3.0)
- CMake (version >= 3.0.2)

### Building

1. `mkdir -p build`
1. `cd build`
1. `cmake ..`
1. `make`

### Running Tests

Still in `build/`, execute `tests/Test` to run test binary.
